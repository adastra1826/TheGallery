Lab notes · dispatches from Studio 7F

Work in progress.

  1. 08.22.49

    Polishing the mercury

    We rebuilt the hero blob’s environment map eleven times this month. Chrome is not a color; it is everything the room is willing to admit about itself. The room is now mostly softboxes.

  2. 08.09.49

    Gel is a material, not an effect

    HC-003’s buttons got denser this week. A gel button should look refrigerated — cold, wet, slightly too full. If it reads as glass, we failed. Glass doesn’t wobble.

  3. 07.30.49

    No restocks means no restocks

    Four hundred emails about Airport 2000. The edition is closed. A drop that returns is just inventory, and inventory is what 2009 did to the internet.

  4. 07.14.49

    Numbering as a promise

    Every object receives a catalog ID before it receives a name. HC-011 already exists on a whiteboard. The number is a promise that we will finish it.
